ELEMENTS OF HTML

Three Types of ELEMENTS

1. BLOCK LEVEL ELEMENT

2. INLINE-BLOCK ELEMENTS

3. INLINE ELEMENTS

BLOCK LEVEL ELEMENTthe tag which occupies whole width and creates a new line
EX: h1-h6,P, div,semantic elements

INLINE-BLOCK ELEMENTSthe tag which occupies the width of its content
EX: span, img, a, strong, em, label, input, button, select, textarea

INLINE ELEMENTSthe tag which does not create a new line
EX: b, strong, i, em, u, s, del, ins, sub, sup, code, var, samp, kbd, pre, abbr, acronym, mark, ruby, rt, rp,rtc, rtc, cite, q, time, del, ins, a, img, br, hr, link, script, style, meta, title, base, head, body, html
